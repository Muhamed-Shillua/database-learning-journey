# Database Learning Journey (10 Days)

A structured 10-day learning repository focused on mastering **Databases and SQL** through a practical, hands-on approach.

This repository is designed as a personal learning system that combines theory, exercises, and real problem-solving to build strong database fundamentals.

It is inspired by structured study from the **Information Technology Institute (ITI)** database course and reorganized into a simplified 10-day roadmap for consistent practice and revision.

---

## Purpose

This repository is designed to:

- Learn databases from fundamentals to advanced concepts
- Strengthen SQL skills through continuous practice
- Apply real-world database concepts in practical scenarios
- Improve logical thinking and problem-solving skills
- Build a strong foundation for backend development

---

## Learning Approach

This journey follows an **80% Practice / 20% Learning (Consuming)** approach:

- **80% Practice:** Writing SQL queries, solving tasks, and applying concepts
- **20% Learning:** Reading notes, understanding theory, and reviewing concepts

The focus is on learning by doing rather than passive reading.

---

## Curriculum Overview

This roadmap covers core database concepts from basics to advanced topics:

- Database fundamentals and design principles
- SQL basics and query writing
- Relationships and joins
- Database normalization
- Aggregations and data analysis
- Advanced SQL concepts and optimization
- Stored procedures and triggers
- Database performance optimization
- Basic administration (backup, restore, jobs)

---

## Goal

The goal of this journey is to move beyond SQL syntax and develop a **deep understanding of how databases work in real-world systems**, while building consistent hands-on practice habits.

---

## Final Note

Consistency is the key to mastery.
Small daily progress leads to strong long-term results.

Build daily. Practice consistently. Think in systems, not just queries.
